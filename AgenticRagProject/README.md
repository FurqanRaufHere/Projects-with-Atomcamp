## Project

This repository contains a minimal example app used during the Agentic Bootcamp. It demonstrates loading simple service data from a text file and exposing a lightweight interface (via `app.py`) for retrieval or demonstration purposes.

Goals:
- Provide a compact example of a RAG-style workflow using local text files
- Show minimal project structure for teaching and experimentation
